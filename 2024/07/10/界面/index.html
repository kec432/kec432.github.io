软件安装方法

vmware

许可证: JU090-6039P-08409-8J0QH-2YR7F

phpstudy
建议安装路径不要修改，默认安装在D盘

java22
记得选择安装路径，要求安装路径记得

python3.9

  1. 加入path，自定义安装
    ![[Pasted image 20240708154242.png]]
  2. 全部勾选
    ![[Pasted image 20240708154324.png]]
  3. 前四个全勾，并选择安装位置
    ![[Pasted image 20240708154436.png]]
  4. 注意自定义安装位置
  5. 打开系统环境变量
    ![[Pasted image 20240708154643.png]]
    ![[Pasted image 20240708154656.png]]
  6. 将python加入系统环境变量(复制粘贴)
    ![[Pasted image 20240708154742.png]]
  7. 确定，确定，重启电脑

IDEA

  1. 选择路径安装idea，全勾
  2. 安装完成,关闭idea，以管理员身份运行激活包
  3. 激活完毕，重启电脑
  4. 输入激活码

bandzip(解压软件)

  1. 正常安装，记得选择安装路径
  2. 破解
  3. 去广告，选项-设置

vscode插件

  1. codeium

  2. chinese

  3. code runner

  4. ctrl+滚轮(调节字体大小)

注：

  1. 安装路径不要有中文
  2. 最好不装在C盘
  3. 安装路径不要有空格和符号(_除外)
